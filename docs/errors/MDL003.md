# MDL003 - Invalid DateTime Format

## Description

Use ISO 8601 datetime format.

## Category

Model Errors

## Severity

`error`

## Common Causes

- Input data does not meet validation requirements

## How to Fix

Use ISO 8601 datetime format.

## Example

```json
"validFrom": "2024-01-01T00:00:00Z"
```
