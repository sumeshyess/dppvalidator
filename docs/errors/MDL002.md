# MDL002 - Invalid URL Format

## Description

Provide a valid URL with scheme (https://)

## Category

Model Errors

## Severity

`error`

## Common Causes

- Input data does not meet validation requirements

## How to Fix

Provide a valid URL with scheme (https://)

## Example

```json
"id": "https://example.com/credentials/dpp-001"
```
