# JLD001 - Missing Context

**Severity:** Error
**Layer:** JSON-LD Semantic

## Description

The `@context` property is missing or invalid. DPPs must include a valid JSON-LD
context to ensure semantic interoperability.

## Example

```json
{
  "type": ["DigitalProductPassport", "VerifiableCredential"],
  "credentialSubject": {
    "id": "https://example.com/product/123"
  }
}
```

**Error:** `@context` must be present and valid.

## Resolution

Add a valid `@context` property referencing the UNTP DPP vocabulary:

```json
{
  "@context": [
    "https://www.w3.org/ns/credentials/v2",
    "https://test.uncefact.org/vocabulary/untp/dpp/0.6.1/"
  ],
  "type": ["DigitalProductPassport", "VerifiableCredential"],
  "credentialSubject": {
    "id": "https://example.com/product/123"
  }
}
```
