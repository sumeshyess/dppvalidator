## [0.1.0] - 2026-01-29

### Added

- Initial release of dppvalidator
- Three-layer validation (Schema, Model, Semantic)
- UNTP DPP 0.6.1 schema support
- Pydantic v2 models for DPP entities
- JSON and JSON-LD exporters
- CLI with validate, export, and schema commands
- Plugin system for custom validators and exporters
- External vocabulary support (ISO country codes, UN/CEFACT unit codes)
- Property-based testing with Hypothesis
- Performance benchmarks
- Comprehensive documentation
